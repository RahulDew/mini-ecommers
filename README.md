This is Mini E-Commers WebApp

Tech Used: MongoDB, Reactjs, Nodejs, Expressjs

Other library used in react:
react-icons (for adding icons)
react-hook-form (for form validation)
react-router (routung in react)

To run application

for client: 
npm install
npm run dev

for server:
npm install
npm run dev


FEATURES
  Role based Authentication
  Products features and operations (based on permission)
